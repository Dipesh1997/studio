# **App Name**: VoiceOver Studio

## Core Features:

- Text-to-Speech Generation: Convert user-inputted text into a natural-sounding voiceover, leveraging offline voice models.
- Video Selection: Allow users to select a video file from their device to add a voiceover to.
- Voiceover Integration: Enable users to integrate generated voiceovers with selected video files seamlessly, offline.
- Preview: Allow users to preview the video with the integrated voiceover.
- Video Export: Export the final video file with the integrated voiceover for sharing or further editing.

## Style Guidelines:

- Primary color: Dark blue (#2C3E50) to establish a professional and modern aesthetic.
- Background color: Light gray (#ECF0F1) to ensure contrast and readability, suitable for a dark color scheme.
- Accent color: Sky blue (#3498DB) to highlight interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern, objective feel. Easy to read.
- Use minimalist vector icons for UI elements. Blue to indicate interaction capabilities.
- Clean and intuitive layout with clear separation of text input, video selection, and preview sections.
- Subtle animations to indicate processing or transitions between different states.
